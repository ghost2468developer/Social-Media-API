# Social Media API 🐦💬

A simple **social media backend API** built with **Node.js, Express, Prisma, and JWT authentication**.  
Supports **user registration, login, posts, likes, and comments**.

---

## Features

- User **registration** and **login** with JWT authentication
- Create, read, **like**, and **unlike posts**
- Add and view comments on posts
- Track **likes count** and **comments count** per post
- Fully built with **Prisma ORM** and **SQLite** (easy to switch to Postgres)

---

## Tech Stack

- **Node.js** & **Express** – server and routing  
- **Prisma** – database ORM  
- **SQLite** – lightweight database  
- **JWT** – authentication  
- **bcrypt** – password hashing  
- **CORS** & **dotenv** – environment configuration
